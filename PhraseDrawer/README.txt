Quick explanation in 3 steps of the installation process:

1. Download the Script and PSD file(.jsx file and .psd) you want in skripts folder:
Win: C> Program Files> Adobe Photoshop CC [YourVersion]> Presets> Scripts>[place in a new folder named as you wish]

2. For using that Skript you need alphabet.psd. Open it. 

3. You'll find then the Script available in the File > Scripts > Browse>[folder named as you wish earlier]

The Script does not support any digits exept latin letters and numbers.
